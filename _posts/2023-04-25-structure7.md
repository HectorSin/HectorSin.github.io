### 선택 정렬 알고리즘

```python
def selection_sort(A):
    n = len(A)
    for i in range(n-1):
        least = i
        for j in range(i+1, n):
            if (A[j]<A[least]):
                least = j
        A[i], A[least] = A[least], A[i]
        printStep(A, i + 1)
```

### 삽입 정렬

```python
def insertion_sort(A):
    n = len(A)
    for i in range(1, n):
        key = A[i]
        j = i-1
        while j>=0 and A[j] > key:
            A[j+1] = A[j]
            j -= 1
        A[j+1] = key
        printStep(A, i)
```



### 버블 정렬

```python
def bubble_sort(A):
    n = len(A)
    for i in range(n-1, 0, -1):
        bChanged = False
        for j in range(i):
            if (A[j]>A[j+1]):
                A[j], A[j+1] = A[j+1], A[j]
                bChanged = True
                
        if not bChanged:
            break
        printStep(A, n-i)
```
